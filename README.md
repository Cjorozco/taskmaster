# TaskMaster

Este repositorio es un proyecto educativo creado con fines de práctica y aprendizaje de desarrollo móvil con React Native y Expo.

## Descripción

TaskMaster es una aplicación sencilla de gestión de tareas que permite a los usuarios organizar sus actividades diarias. El objetivo principal de este proyecto es explorar y aplicar conceptos fundamentales de desarrollo de aplicaciones móviles, incluyendo navegación, gestión de estado y consumo de APIs.

## Características

- **Listado de Tareas**: Visualización de todas las tareas pendientes y completadas.
- **Detalle de Tarea**: Vista detallada de la información de una tarea específica.
- **Creación y Edición**: Formularios para agregar nuevas tareas o modificar las existentes.
- **Gestión de Estado**: Uso de herramientas modernas para el manejo del estado de la aplicación.

## Tecnologías Utilizadas

- React Native
- Expo
- React Navigation
- TypeScript

## Nota sobre la API

Este proyecto consume [JSONPlaceholder](https://jsonplaceholder.typicode.com/todos), una API pública y gratuita para pruebas. 
**Importante**: Como es una API "fake", los cambios (crear, editar, eliminar) **no persistirán** en el servidor real. La aplicación simula estas operaciones para demostrar el flujo de trabajo, pero los datos se restablecerán o no se reflejarán si se recarga desde el servidor.

## Enfoque Educativo

El código fuente contiene **comentarios detallados** explicativos. Estos comentarios tienen como objetivo:
1. Explicar el "por qué" de ciertas decisiones técnicas.
2. Describir el funcionamiento general de componentes y hooks.
3. Servir como guía de lectura para quienes están aprendiendo React Native.
